Janky Hollow Knight Project

For the final project in my ICS4U1 course, I was provided with a template of a basic, unfinished Zelda game and tasked with creating something new using it. The project was designed to help me learn how to navigate unfamiliar documentation and work in a new IDE—Greenfoot—which none of us had used before. This experience taught me how to efficiently learn a new programming language while allowing for a significant amount of creative freedom. It was a great opportunity to develop my problem-solving skills and explore new aspects of game development.
